📊 Samsung Supply Chain & Logistics Dashboard

This project presents a comprehensive supply chain and logistics dashboard designed for Samsung. 

It provides real-time insights into suppliers, manufacturing, shipments, and customer data, enabling better decision-making and performance tracking.


🚀 Key Features

- Financial Overview: Displays gross revenue, total revenue, profit, and profit margin.
- Operational Metrics: Tracks order quantity, inventory levels, shipment volume, and delivered quantity.
- Supplier Analysis: Compares lead times across suppliers to identify efficiency.
- Inventory Stock: Monitors product-wise stock availability (Galaxy series, wearables, etc.).
- Shipment Performance: Highlights carriers with the most delayed shipments.
- Revenue Sources: Breaks down revenue contribution by platforms (Amazon, Flipkart, Best Buy, etc.).
  
🎯 Purpose

This dashboard is built to:
- Enhance visibility across the supply chain.
- Identify bottlenecks in shipments and supplier lead times.
- Provide data-driven insights for strategic decision-making.
- Support real-time monitoring of logistics and customer fulfillment.

  
📈 Tech Stack


- Data Visualization: Interactive charts, bar graphs, and progress indicators.
- Dashboard Design: Clean, organized layout with intuitive navigation.
- Use Case: Ideal for supply chain managers, analysts, and logistics teams.
